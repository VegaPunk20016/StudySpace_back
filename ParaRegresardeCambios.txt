Rutas de los endpoints:

Usuarios:

POST:
-----> api/usuarios 
      (crea un usuario)
       {
        "nombre":"",
        "correo":"",
        "password":""
       }
-----> api/usuarios/login
       {
        "correo":"",
        "password":""
       }

GET:
-------> api/usuarios 
         (trae todos los usuarios)
-------> api/usuarios/:id
         (trae usuario por id)

PUT:
-------> api/usuarios/:id
         (modifica usuario por id)
        {
        "nombre":"",
        "correo":"",
        "password":""
        }

DELETE:
-------> api/usuarios/:id
         (borra usuario por id)




Materias:

POST:
-----> api/materias 
      (crea una materia)
       {
        "nombre":"",
        "descripcion":""
       }

GET:
-------> api/materias 
         (trae todas las materias)
-------> api/materias/:id
         (trae materia por id)
-------> api/materias/search
         (esta ruta funciona como /search?nombre=mate
          nota: cuidado, si tu materia tiene un caracter especial como +
          conviertelo a codificacion URL o URL encoding)

PUT:
-------> api/materias/:id
         (modifica materia por id)
        {
        "nombre":"",
        "descripcion":""
        }

DELETE:
-------> api/materias/:id
         (borra materias por id)




Sesiones

POST:
-----> api/sesiones 
      (crea una sesion)
       {
        "usuario_id": ,
        "materia_id": ,
        "fecha":"",
        "duracion_min":
       }
       (nota: la fecha tiene que estar en formato
        ISO8601)

GET:
-------> api/sesiones
         (trae todas las sesiones)
-------> api/sesiones/:id
         (trae sesion por id)
-------> api/sesiones/usuario/:usuario_id
         (busca sesiones por usuarios)
-------> api/sesiones/usuario/:usuario_id/paginated
         (busca sesiones por usuarios y las pagina)

PUT:
-------> api/sesiones/:id
         (modifica sesion por id)
        {
         "usuario_id": ,
         "materia_id": ,
         "fecha":"", 
         "duracion_min":
        }

DELETE:
-------> api/sesiones/:id
         (borra sesiones por id)
